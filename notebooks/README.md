# Notebooks

This folder contains two versions of the project notebook:

- **`NHS_Waiting_Time_original.ipynb`** — the original notebook file uploaded by the author.
- **`NHS_Waiting_Time_clean_for_github.ipynb`** — a cleaned public-facing version with outputs cleared and a few scratch/error cells removed for a tidier GitHub presentation.

## Important note

The cleaned notebook preserves the project workflow, but the repository still does **not** contain every original raw NHS extract used during the full analysis process. The included outputs and tables match the packaged results, but full end-to-end reruns may require re-downloading NHS source files referenced in the notebook.
